Тестовое задание для Центра программного обеспечения. Выполнено Снятковым Дмитрием Андреевичем.

Для развертки понадобится docker, make и возможно VPN (нынче регистры докера барахлят).

1. Создать .env файл и подредактировать его под вашу систему
2. Вставить корректные данные в phpunit.xml (опционально, если будете запускать тесты)
3. make install (это запустит очередь и засидит проект для удобства)

Postman - https://www.postman.com/solar-escape-122205/swctest

Примечание:
Не знаю специально или нет, но задача немного неккоретна - нельзя в Laravel делать multiform data через PUT. Я с таким 
встречался, поэтому поменял роутинг для редактирования задачи.
Политики решил не добавлять, так как в задаче не подразумевается кто и при каких услових может работать с задачами.
